Редактор кампаний

С помощью Редактора кампаний вы сможете устанавливать основные параметры создаваемой вами кампании, определять интерфейс экрана загрузки, включать в кампанию нестандартные объекты и импортированные файлы.
 

Основные параметры

Название - название кампании.
Рекомендуется игроков - для каких типов игр и конфигураций оптимизирована ваша кампания.
Автор - ваше имя (прозвище).
Описание - описание вашей кампании.
Уровни сложности - при включенном параметре ваша кампания может быть дифференцирована по уровню сложности. Это достигается через обращение к условию "Game Difficulty Comparison" в Редакторе триггеров.
Файлы карт - все карты, которые были использованы в вашей кампании.
 

Экран загрузки

Здесь можно создать интерфейс экрана загрузки: указать фоновый экран, позиционный звук, а также внешний вид кнопок на разных картах кампании.

Идентификатор - к этому числу может обращаться действие "Show/Hide Custom Campaign Button" в Редакторе триггеров.
Видимая - будет ли видима кнопка при первом доступе к кампании.
Загруженный файл - название карты, соответствующей кнопке.
Глава - заголовок главы ("Пролог", "Глава 1" и т.п.).
Название - название главы.
Фоновый экран - фон экрана кампании.
Позиционный звук - звуки на экране кампании.
 

Нестандартные данные

Каждая карта кампании имеет доступ ко всем объектам, созданным либо существующим в Редакторе объектов: ко всем войскам, предметам, декорациям, способностям и исследованиям. За более подробными сведениями по этому вопросу обратитесь к разделу Редактор объектов.
 

Импортированные файлы

Каждая карта кампании имеет доступ ко всем импортированным файлам, находящимся в менеджере импорта, то есть все эти файлы можно использовать на картах кампании. За более подробными сведениями по этому вопросу обратитесь к разделу Менеджер импорта.
